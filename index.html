Security Architecture

Built with Security in Mind

Prototype-level security with clear markers showing where production-grade measures belong.

  • 🔒

    bcrypt Password Hashing

    All passwords are hashed with bcrypt before storage. Plain text is never stored.

  • 🎲

    Cryptographic Share Codes

    Share codes use Math.random seeded generation — in production, use crypto.randomBytes().

  • ⏱

    Code TTL Expiry

    Codes automatically expire after a set duration. Expired codes are rejected silently.

  • ✔

    Input Validation

    All forms validate email format, password strength, and field presence before submission.

  • 🧩

    Session Management

    Login state persisted via sessionStorage. In production, use secure HttpOnly cookies.
